# Course Graphics Design System

## Purpose

This document defines the visual design standards for all programmatically generated graphics in the MBA Strategy Course. All graphics are generated by Python tools using matplotlib, following these specifications for consistency across ~63 figures.

---

## Color Palette (Modern/Bold)

| Color | Hex | Usage |
|-------|-----|-------|
| Navy Blue | `#1B2A4A` | Primary headers, dark backgrounds, matrix labels |
| Steel Blue | `#3D5A80` | Secondary backgrounds, axis lines, borders |
| Deep Teal | `#2A9D8F` | Positive/growth regions (Stars, Grow/Build, Aggressive) |
| Warm Gold | `#E9C46A` | Highlights, key labels, neutral/maintain regions |
| Burnt Orange | `#E76F51` | Caution items (Question Marks, action-required) |
| Muted Red | `#C1292E` | Negative/threat regions (Dogs, Harvest/Divest, Defensive) |
| Off-White | `#F8F9FA` | Figure background |
| Light Gray | `#E9ECEF` | Grid lines, subtle borders, alternating rows |
| Near-Black | `#212529` | Primary text on light backgrounds |
| Dark Gray | `#495057` | Secondary text |
| White Text | `#F8F9FA` | Text on dark backgrounds |

### Region Color Mapping

| Strategic Region | Color | Opacity |
|-----------------|-------|---------|
| Grow/Build/Aggressive/Strong | Deep Teal | 20% fill |
| Hold/Maintain/Competitive/Neutral | Warm Gold | 20% fill |
| Harvest/Divest/Defensive/Weak | Muted Red | 20% fill |
| Conservative/Parity | Light Gray | 100% fill |

| Element | Font | Size | Weight |
|---------|------|------|--------|
| Figure number | Segoe UI / Arial | 13pt | Bold |
| Figure title | Segoe UI / Arial | 12pt | Italic |
| Axis labels | Segoe UI / Arial | 11pt | Semibold |
| Quadrant/cell labels | Segoe UI / Arial | 11pt | Bold |

| Annotations/notes | Segoe UI / Arial | 8pt | Regular |

Font stack: `Segoe UI` (Windows), `Arial` (fallback), `sans-serif` (universal fallback)

## Layout Rules

| Property | Value |
|----------|-------|
| Default figure size | 10 x 7 inches (landscape) |
| Tall diagram size | 8 x 10 inches (portrait) |
| DPI | 200 |
| Background color | Off-White `#F8F9FA` |
| Content border | 1pt Light Gray `#DEE2E6` |
| Title area | Reserved at top of figure, 0.8 inches |
| Content margins | 0.6 inches all sides below title |

---

## Figure Numbering Convention (GCU Compliant)

Format: `Figure {topic}.{sequence}`

- Topic number matches the chapter (1, 2, 3, 4, 6, 7)
- Sequence resets per topic (1.1, 1.2, ... 3.1, 3.2, ...)
- Tool guide figures: `Figure TG.{sequence}`

### GCU Label Format

Rendered inside the image at the top:

```
Figure 3.1                  ← Bold, 13pt
VRIO Decision Tree           ← Italic, 12pt
```

Both lines centered. No period at end. No colon.

## How to Regenerate

```bash
python tools/generate_graphics.py              # All graphics
python tools/generate_graphics.py --topic 3    # One topic
python tools/generate_graphics.py --id vrio    # One figure
python tools/generate_graphics.py --verify     # Check outputs
```

Change any value in `tools/graphics/config.py`, re-run, and all graphics update.
